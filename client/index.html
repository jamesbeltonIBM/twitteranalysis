This web site has been developed using the IBM Cloud. The application gathers tweets and uses sentiment analysis powered by IBM Watson to give each tweet a score, based on the 'positive' and 'negative' words used in each tweet. This is a powerful way of determining how the users of Twitter are feeling about a subject, product, brand or organisation.
